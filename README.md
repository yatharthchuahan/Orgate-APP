# Oragtre Flutter App

A mobile application built with **Flutter** for managing inventory, sales, and user accounts.  
Supports Android, iOS, web, Linux, macOS, and Windows platforms.

---

## 📁 Project Structure

```
lib/
├─ main.dart
├─ model/             # Data models (responses, requests)
├─ Screens/           # UI screens (login, home, inventory, etc.)
├─ Services/          # API & business logic services
├─ StoredData/        # Local storage helpers
├─ utils/             # Utility functions & helpers
└─ Widgets/           # Reusable UI components
android/ ios/ web/…    # Platform-specific code generated by Flutter
```

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd Oragtre_final_app-master
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

   - Choose a connected device or emulator.
   - For a specific platform, use `-d <device-id>` (e.g., `flutter run -d chrome`).

---

## 🛠️ Features

- User authentication (login, password recovery)
- Inventory management and stock updates
- Sales tracking and update functionality
- Profile handling with picture uploads
- Cross-platform support (mobile, web, desktop)

---

## ⚙️ Utilities & Helpers

- **Authentication**: `auth_handler.dart`, `perform_logout.dart`
- **Networking**: `network_utils.dart`
- **Data formatting**: gender, time, base64 conversion
- **Persistent storage**: user/login helpers
